⚙️ Settings

Configure The Moment and manage your printers

📚 Getting Started

  1. Spoolman: Spoolman is included in The Moment's docker-compose.yml and starts automatically alongside it. If you have an existing Spoolman instance, point The Moment at it via Settings → Basic Configuration → Spoolman URL. Add your filament spools in Spoolman before proceeding.
  2. Add Printers: Go to Settings → Printers and click "Add Printer". Supported interfaces: OctoPrint (Ender 3 and similar), PrusaLink (Prusa printers), and Bambu (X1C, P1S, A1, etc. via MQTT on LAN). A virtual test printer is also available for testing without hardware.
  3. Assign Spools to Toolheads: Go to the Filament Status tab and assign a Spoolman spool to each toolhead. This tells The Moment which filament is loaded.
  4. Start Printing: The Moment polls your printers automatically. When a print finishes, filament usage is logged and written back to Spoolman.
  5. NFC Tags (optional): Go to the NFC Management tab to generate tag files for your spools and printer toolhead locations. Write them to NTAG213 stickers using NFC Tools. Scanning a tag on your phone opens the assignment page directly.

🖨️ Per-Printer Overrides

Override global defaults for each printer. Zero means "use global". Print wattage replaces the global default. Preheat adds a one-time electricity charge per print. High-temp extra wattage is added automatically when Spoolman identifies the spool material as ABS, ASA, PA, or PC. Purchase cost ÷ lifespan derives the per-hour depreciation; or enter it directly.

🏷️ NFC Spool Locations

These Spoolman location names are used by the NFC spool-tag workflow. When a spool is displaced from a toolhead the old spool moves to Inventory automatically. When a spool is finished, tapping its tag and choosing Trash moves it to Trash so the tag can be re-programmed. Both locations are created in Spoolman automatically the first time they are used.
When enabled, assigning a spool to a toolhead in Print Ops writes its location to Spoolman as {Printer Name} - T{index} (e.g. "Ender3 - T0"). The Moment also polls Spoolman every 5 minutes and updates toolhead assignments when you move a spool's location there. Unassigning returns the spool to the Inventory location below. Changes save immediately.
