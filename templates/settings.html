📚 Getting Started

  1. Install Spoolman: Run docker run -d --name spoolman -p 8000:8000 -v spoolman-data:/home/spoolman/data ghcr.io/donkie/spoolman:latest
  2. Add Printers: Click "Add Printer" in the Printers tab and enter your printer's hostname or IP address and API key
  3. Map Spools: Go to the "Filament Status" tab and assign spools to toolheads
  4. Start Printing: The system will automatically track filament usage!

🖨️ Per-Printer Overrides

Override global defaults for each printer. Zero means "use global". Print wattage replaces the global default. Preheat adds a one-time electricity charge per print. High-temp extra wattage is added automatically when Spoolman identifies the spool material as ABS, ASA, PA, or PC. Purchase cost ÷ lifespan derives the per-hour depreciation; or enter it directly.
